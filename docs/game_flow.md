# Game Flow

# Overview

The Guess The Number game is an interactive embedded system application built using Arduino Uno, a 4×4 keypad, a 16×2 LCD display, and LED indicators.
The game allows a user to guess a secret number generated by the system or set manually by another user. The Arduino processes each guess and provides hints through the LCD display and LEDs until the correct number is found.

# Game Modes

The system provides two game modes:

Key	Mode	Description
A	Random Mode	The Arduino generates a random number automatically

B	Manual Mode	User 1 enters the secret number manually

This allows flexibility for both single-player and two-player gameplay.


# Game Flow Steps

The following steps describe the overall game process.

Step 1 – System Initialization

The Arduino system powers ON.

LCD display initializes and shows the welcome message.

All LEDs remain OFF.

Step 2 – Mode Selection

The user selects the game mode using the keypad.

Press A → Random number mode

Press B → Manual number mode

Step 3 – Secret Number Setup

Random Mode

Arduino generates a random number within the defined range.

The number is stored internally.

Manual Mode

User 1 enters the secret number using the keypad.

The number is stored in memory.

Step 4 – Guess Input

User enters a guess using the keypad.

The Arduino reads the entered number.

The number is compared with the stored secret number.

Step 5 – Hint Generation

The system analyzes the difference between the guessed number and the secret number and provides feedback.

Condition	               LCD Message	     LED Indication

Guess > Secret	         Lower	           Red LED ON

Guess < Secret	         Higher	           Red LED ON

Guess close to number	   So Close	         Yellow LED ON

Guess = Secret	         Correct	         Green LED ON

Step 6 – Game Loop

If the guess is incorrect, the system asks the user to try again.

The user continues entering guesses until the correct number is found.

Step 7 – Game Completion

When the correct number is guessed:

LCD displays “Correct!”

Green LED turns ON.

The game ends or resets for a new round.

# Conclusion

The game flow ensures smooth interaction between the user input (keypad), processing unit (Arduino), and output devices (LCD and LEDs). The structured logic allows users to receive hints and progressively refine their guesses until the correct number is identified.
