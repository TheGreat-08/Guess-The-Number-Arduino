## Testing Results

The system was tested by entering different guessed numbers using the keypad. The Arduino compared each input with the generated random number and displayed the appropriate result.

| Test No | Random Number | User Guess | Result  |
| ------- | ------------- | ---------- | ------- |
| 1       | 5             | 3          | Higher  |
| 2       | 5             | 7          | Lower   |
| 3       | 5             | 5          | Correct |
| 4       | 8             | 6          | Higher  |
| 5       | 8             | 9          | Lower   |
| 6       | 8             | 8          | Correct |

The testing results confirm that the system correctly identifies whether the guessed number is higher, lower, or equal to the generated random number.
