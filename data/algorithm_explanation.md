## Algorithm Explanation

1. Start the system.
2. The Arduino generates a random number between 1 and 10.
3. The user enters a guess using the input buttons.
4. The system compares the guessed number with the random number.
5. If the guess is correct, the system indicates success using LED.
6. If the guess is higher than the random number, a "Lower" indication is given.
7. If the guess is lower than the random number, a "Higher" indication is given.
8. The process continues until the correct number is guessed.
9. Stop the program.
